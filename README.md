## üíª How To Install?

- Clone This Repo

```
git clone https://github.com/easyManik/belanjaApp-frontend.git
```

- Go To Folder Repo

```
cd belanjaApp-frontend
```

- Install Module

```
npm install
```

- Type ` npm start` To Start Website
